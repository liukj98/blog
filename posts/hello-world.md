# 你好，世界

这是博客的第一篇文章。

## 为什么写博客

写博客是一种很好的学习方式，通过输出倒逼输入，把学到的知识整理成文字，加深理解的同时也能帮助到其他人。

## 技术栈

本站使用以下技术搭建：

- **VitePress** - 基于 Vite 的静态站点生成器
- **Vue 3** - 渐进式 JavaScript 框架
- **Markdown** - 轻量级标记语言
